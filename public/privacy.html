마작기록장 개인정보 처리방침

'마작기록장'(이하 '서비스')은(는) 사용자의 개인정보를 중요시하며, "개인정보보호법" 등 관련 법령을 준수하고 있습니다. 본 처리방침을 통해 사용자가 제공하는 개인정보가 어떠한 용도와 방식으로 이용되고 있는지 알려드립니다.

1. 수집하는 개인정보 항목

서비스는 원활한 서비스 제공을 위해 아래의 최소한의 개인정보를 수집합니다.

  • 필수 항목: Google 계정 이메일
  • 수집 방법: Google 로그인(Firebase Authentication 기반)

2. 개인정보 및 권한의 수집/이용 목적

수집한 개인정보는 다음의 목적을 위해서만 활용됩니다.

  • 사용자 인증 및 로그인 상태 유지
  • 마작 게임 기록의 저장, 조회, 수정, 삭제 기능 제공
  • 공유된 기록장 접근 및 데이터 동기화
  • 서비스 운영, 오류 대응 및 부정 사용 방지
[Google 로그인 및 데이터 처리 안내]
본 서비스(MahjongSheet)는 사용자의 Google 계정을 이용해 로그인 기능을 제공합니다. 로그인에 사용되는 정보는 사용자 식별 및 서비스 접근 제어를 위한 최소 범위에서만 사용됩니다.
사용자가 입력한 게임 기록 데이터는 서비스 기능 제공을 위해 Firebase Authentication 및 Cloud Firestore 기반 인프라를 통해 저장·동기화되며, 개발자의 외부 서버로 무단 전송되거나 제3자에게 판매되지 않습니다.

[Google Sign-In and Data Handling]
MahjongSheet uses Google Sign-In to authenticate users. The Google account information used for sign-in is processed only to the minimum extent necessary for user identification and access control.
Game record data entered by users is stored and synchronized using Firebase Authentication and Cloud Firestore solely for service functionality, and is not transferred without authorization to external developer-operated servers or sold to third parties.

3. 데이터 보호 조치

서비스는 사용자의 개인정보 및 서비스 데이터를 안전하게 보호하기 위해 다음과 같은 조치를 적용하고 있습니다.

  • 전송 구간 암호화: 사용자 기기와 서비스 인프라 간의 데이터 전송은 HTTPS 기반 암호화 통신을 사용합니다.
  • 접근 통제: 인증된 사용자만 자신의 기록 또는 공유받은 기록장 데이터에 접근할 수 있도록 접근 제어를 적용합니다.
  • 보안 인프라 사용: 사용자 인증 및 데이터 저장에는 Firebase Authentication 및 Cloud Firestore와 같은 관리형 보안 인프라를 사용합니다.

[Data Protection Mechanisms]
To protect user data, MahjongSheet applies the following safeguards:
- Encryption in Transit: Data transmitted between the user's device and the service infrastructure is protected via HTTPS.
- Access Control: Only authenticated users can access their own or explicitly shared records.
- Managed Security Infrastructure: Firebase Authentication and Cloud Firestore are used for secure authentication and data storage.

4. 개인정보의 보유 및 이용 기간

원칙적으로, 개인정보 수집 및 이용 목적이 달성되거나 사용자가 회원 탈퇴를 요청할 경우 해당 개인정보를 지체 없이 파기합니다. 단, 관계 법령의 규정에 의하여 보존할 필요가 있는 경우 해당 법령에서 정한 기간 동안 보관합니다.

5. 개인정보의 제3자 제공 및 공유 금지

서비스는 사용자의 사전 동의 없이 개인정보를 외부에 제공하지 않습니다. 단, 법령의 규정에 의거하거나 수사 목적으로 법령에 정해진 절차와 방법에 따라 수사기관의 요구가 있는 경우는 예외로 합니다.

[데이터 공유 금지]
본 서비스는 사용자의 개인정보 및 서비스 이용 데이터를 광고, 마케팅, 데이터 브로커 등 제3자에게 공유하거나 판매하지 않습니다.

[No Sharing of User Data]
We do not share, transfer, or sell users' personal information or service usage data to third parties, including for advertising, marketing, or data broker purposes.

6. 사용자의 권리와 그 행사 방법

사용자는 언제든지 서비스 내 탈퇴 또는 데이터 삭제 기능을 통해 자신의 정보에 대한 조회, 수정, 삭제를 요청할 수 있습니다. 또한 사용자는 본인의 Google 계정 설정을 통해 로그인 연동 상태를 관리할 수 있습니다.
